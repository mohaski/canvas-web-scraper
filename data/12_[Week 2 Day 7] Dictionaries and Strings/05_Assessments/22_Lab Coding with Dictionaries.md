# Lab: Coding with Dictionaries

## ![](https://moringa.instructure.com/courses/1391/files/620338/preview?) Lab: Coding with Dictionaries

A **dictionary** in Python is an unordered collection of key-value pairs, where each key is unique and maps to a corresponding value. You can access, add, update, or delete values using their keys, which makes dictionaries ideal for organizing and retrieving data efficiently by name or identifier rather than by position.

**Dictionaries are used when you need to store and access data by a meaningful label rather than a position.** Dictionaries are especially useful for representing structured information, like the attributes of a person, configuration settings, or counts of items. Dictionaries shine in situations where quick lookups are needed, such as when building frequency counters, caching results, or parsing structured input.

In this lab, you are going to be assessed on your ability to:

* Look up a value in a dictionary with a given key.
* Create a dictionary using the given key-value pairs.
* Add a new key-value pair to an existing dictionary.
* Remove a key-value pair from a dictionary.
* Iterate over a dictionary.
