# Lab: Coding with While Loops

## // Lab: Coding with While Loops

A while loop is a way to execute a specified task a dynamic number of times in Python. A while loop repeatedly executes a block of code as long as a specified condition remains True. To prevent infinite loops, it’s important to ensure that the loop’s condition eventually becomes False through some change within the loop.

While loops are typically used when the number of iterations is not known in advance, and when looping depends on dynamic factors during runtime. This could be to continue looping back to a main menu of choices until a user types 'quit' or to continue a math operation until a maximum or minimum is reached.

In this lab, you are going to be assessed on your ability to:

* Identify risks of using while loops.
* Create while loops.
  + Print the numbers from 1 to 5 inclusive.
  + Implement user input to determine the end of the loop.
* Debug while loops.
