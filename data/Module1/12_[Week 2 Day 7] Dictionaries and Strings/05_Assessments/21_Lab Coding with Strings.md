# Lab: Coding with Strings

## ![](https://moringa.instructure.com/courses/1391/files/620338/preview?) Lab: Coding with Strings

A **string** in Python is an immutable sequence of characters used to represent text. Like other sequences in Python, strings support indexing, slicing, and iteration, allowing you to access or manipulate specific characters or portions of the string. However, because strings are immutable, any operation that appears to modify a string (such as replacing characters) actually creates a new string.

**Strings are used to store and manipulate text-based data, making them essential for user input, file handling, communication between programs, and displaying output.** You might use strings to collect names, format messages, or parse and process structured text like CSV lines. Python offers a rich set of string methods, such as `.split()`, `.join()`, `.replace()`, and `.lower()`, to transform and analyze text efficiently.

In this lab, you are going to be assessed on your ability to:

* Access characters in a string by index.
* Create multiline strings.
* Check for the presence of a substring within a string.
* Use f-strings for string interpolation.
* Implement built-in string methods.
