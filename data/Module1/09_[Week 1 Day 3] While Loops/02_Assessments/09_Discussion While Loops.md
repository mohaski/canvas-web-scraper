**Background**

While loops are useful for situations where you need to repeat an action, but the exact number of repetitions isn't known beforehand. The basic structure of a while loop is:

 Here's a basic structure of a while loop:

```
while (condition):  
  # Code to be repeated
```

* `while (condition):` defines when the loop should continue running
* The indented code block is executed repeatedly as long as the condition is true
