# Lab: Coding with Tuples

## ![](https://moringa.instructure.com/courses/1391/files/620338/preview?) Lab: Coding with Tuples

A **tuple** in Python is an immutable, ordered collection of elements. Like lists, tuples can store multiple items, including numbers, strings, and even other collections. However, once a tuple is created, its contents cannot be changed.

**Tuples are used when you want to group data together and guarantee that it won’t change.** Because they are immutable, they are often safer to use than lists in contexts where accidental modification could cause bugs and provide a reliable structure for storing fixed sets of related values like coordinates, RGB color codes, or database records. Tuples are especially useful in function returns when you want to return multiple values at once, offering a clean and structured way to handle related but distinct pieces of data.

In this lab, you are going to be assessed on your ability to:

* Create tuples with given values.
* Print elements in a tuple.
* Use tuple methods such as `count` and `index`.
* Check for the presence of a given element in a tuple.
* Implement tuples into functions.
