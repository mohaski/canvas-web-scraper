# Lab: Coding with Operators and Control Flow Statements

## ![](https://moringa.instructure.com/courses/1391/files/620338/preview?) Coding with Operators and Control Flow Statements

Operators and control flow statements are the tools that help your program make decisions and act differently depending on the situation. Allowing your code to respond to conditions, compare values, and decide what to do next.

**Operators** are symbols (like `+`, `>`, or `==`) used to perform operations on values or variables. You’ll use them to do math, check conditions, and assign values.

**Control flow statements**, like `if`, `elif`, and `else`, tell your program which blocks of code to run depending on whether certain conditions are True or False.

These tools work together to help your programs make smart choices. Whether you're checking if a user typed the correct password or deciding whether a score qualifies as a "pass" or "fail", operators and control flow make it possible.

In this lab, you'll practice using operators and control flow to build programs that react and respond to different situations—just like a real app would. You are going to be assessed on your ability to:

* Identify common operator(s)
* Demonstrate that you can implement operators with Python
  + Declare variables
  + Utilize simple operators for specific outputs
* Demonstrate that you can code control flow statements with Python
