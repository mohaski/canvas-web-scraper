Generative AI has emerged as a groundbreaking technology that is transforming the field of data science. As a beginning data science student, it is essential to understand the capabilities and applications of generative AI tools. Generative AI has revolutionized the field of data science by enabling the creation of intelligent systems that can help the programmer create code as well as use it to help debug. While Generative AI can perform other tasks in data science such as creating synthetic data and modeling, the emphasis here is to use generative AI as a “code assistant”. This module will guide you through three topics:

### Introduction to Generative AI

---

We will explore the foundation and terminology of generative AI as well as introduce you to a number of available tools and use cases. You will get a basic understanding of how Generative AI can be used to assist you as a data scientist.

### Prompt Engineering

---

This topic will cover the proper language, context, and setup needed to produce good results when prompting a generative AI model. It is vital to craft and develop prompts that will help you interact with and receive useful answers from the generative AI model.

### Generating Python and SQL Code

---

This topic will dive into using generative AI prompts to help debug and write python code, as well as help you write SQL queries. By understanding the subtleties involved you can increase your own coding efficiencies using these tools.

### Industry Examples

#### JPMorgan Chase - Financial Services Problem

**Organization: Mayo Clinic**

Needed to accelerate analysis of complex financial documents and research reports.

**Solution:** Implemented generative AI to automatically extract key insights from financial documents, summarize research, and generate preliminary analysis reports. This reduced analyst research time by 30% and improved consistency of analysis.

#### Moderna - Pharmaceuticals Problem

Time-consuming process of analyzing protein sequences for vaccine development.

**Solution:** Used generative AI models to predict protein structures and interactions, significantly accelerating the initial screening process for potential vaccine candidates. This helped reduce early research phase timelines by approximately 40%.

#### Walmart - Retail Problem

Manual effort required to optimize product descriptions and search relevance.

**Solution:** Deployed generative AI to automatically create and optimize product descriptions, tags, and search metadata across millions of items. This improved search accuracy by 25% and reduced manual content creation time by 60%.

Each example demonstrates how generative AI can:

* Automate repetitive analysis tasks
* Extract insights from unstructured data
* Accelerate decision-making processes
* Scale data science operations efficiently

### Learning Outcomes

After completing this module, you should be able to:

* Identify the major components of Generative AI, including Large Language Models (LLMs), neural networks, and tokenization, by analyzing industry applications.
* Apply prompt engineering techniques to optimize interactions with generative AI tools for coding and debugging in Python and SQL.
