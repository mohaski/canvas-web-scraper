### Debugging Assistance

1. When encountering errors or bugs in the code, feed the error messages, stack traces, or relevant code snippets to the generative AI model.
2. Allow the model to analyze the provided information and suggest potential fixes, improvements, or explanations for the encountered issues.
3. Evaluate the model's suggestions and apply the most appropriate fixes to resolve the debugging challenges.

### Generating Code

### 1. Requirement Analysis & Planning

* Define business problem/needs.
* Identify input data sources.
* Determine expected outputs.
* List technical constraints.
* Set success criteria.

### 2. Development Strategy

* Break down into components.
* Plan data transformations.
* Choose appropriate libraries/tools.
* Define error handling needs.
* Outline testing approach.

### 3. Iterative Code Generation

* Start with basic structure.
* Request specific functionality.
* Add error handling.
* Incorporate validation.
* Refine based on needs.

### 4. Testing & Validation

* Verify data accuracy.
* Test edge cases.
* Check performance.
* Validate business rules.
* Document limitations.

### 5. Refinement & Optimization

* Improve code efficiency.
* Enhance readability.
* Add documentation.
* Optimize performance.
* Implement feedback.
