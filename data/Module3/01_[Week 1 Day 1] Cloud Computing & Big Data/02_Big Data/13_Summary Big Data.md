### Terms Recap

* **Volume:**The massive amount of data generated from various sources.
* **Velocity:** The speed at which data is generated, collected, and processed.
* **Variety:**The diverse types of data, including structured (databases), semi-structured.
* **Data Collection and Storage:** Data is stored in a distributed system (HDFS) across multiple nodes.
* **Defining the Map Phase:** The dataset is broken down into key-value pairs for parallel processing.
* **Shuffling and Sorting:** Key-value pairs are shuffled and sorted to prepare for aggregation.
* **Defining the Reduce Phase:** Aggregation operations are performed to generate meaningful insights.
* **Output Storage:** The final results are saved to distributed storage for further use.
* **Analyzing and Using the Output:** Data is analyzed and applied to business decisions.

### Key Concepts

* **In-Memory Processing:** Processing data directly in memory (RAM) rather than on disk.
* **Real-Time Data Processing:** The ability to process data as it is generated.
* **Fault Tolerance:** The ability to recover from node failures without data loss.
* **Distributed Computing:** The process of distributing tasks across multiple computers (nodes).
