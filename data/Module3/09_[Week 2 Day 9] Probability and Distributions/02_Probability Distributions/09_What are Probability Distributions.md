Probability distributions are mathematical functions that describe how likely values in a dataset are expected to occur. To understand this fully, let's break down the key terms and concepts:

### Probability Distribution

Probability distribution is a **function that gives the probability of each possible outcome in an experiment or random process**. Think of it as a mathematical recipe that tells us how likely different results are to happen.

### Random Variable

Random variables **have a value that is determined by random chance**. Random variables come in two types:

* **Discrete:** Can only take specific, separate values (like counting whole numbers)
* **Continuous:** Can take any value within a range (like measurements that can be fractional)

### Probability Mass Function (PMF)

For discrete random variables, the **PMF tells us the exact probability of each possible value**. For example, when rolling a fair die, the probability of rolling any number is exactly 1/6.

### Probability Density Function (PDF)

For continuous random variables, the **PDF shows the relative likelihood of different values occurring**. Unlike PMFs, we use PDFs to find the probability of values falling within ranges rather than exact values.

### Cumulative Distribution Function (CDF)

The CDF **gives the probability of a value being less than or equal to a specific point**. The CDF helps answer questions like "What's the chance of a response time being under 150 milliseconds?"

### Percent Point Function (PPF)

Also known as the Quantile Function or Inverse Cumulative Distribution Function, the PPF is essentially the inverse of the CDF. While the CDF tells us "what percentage of values fall below x?", the **PPF answers the question "what value of x has y% of the data below it?"**

### Expected Value (Mean, μ)

The expected value is **the average value we expect to see if we repeat an experiment many times**. For example, if we measure response times for a web service, the expected value might be 100 milliseconds.

### Standard Deviation (σ)

Standard deviation is **a measure of how spread out the values are from the mean**. A larger standard deviation means values tend to be more scattered. In our response time example, a standard deviation of 20 milliseconds tells us most values fall between 80 and 120 milliseconds.

### Variance (σ²)

A square of the standard deviation, **variance provides another way to measure spread.** While standard deviation uses the same units as our data, variance uses squared units.

### Parameters

Parameters are **numbers that define the mathematical shape and behavior of a distribution**. Common parameters include:

* **Location parameters** (like the mean) that shift the distribution left or right
* **Scale parameters** (like standard deviation) that stretch or compress the distribution
* **Shape parameters** (like degrees of freedom) that affect the overall form of the distribution and are often related to sample size

### Support

The **set of all possible values that could occur in the distribution** is considered support. For example:

* A **normal distribution** has support from negative infinity to positive infinity.
* A **uniform distribution** between 0 and 1 has support only in that range.

 These concepts work together to give us a complete mathematical description of random processes.
