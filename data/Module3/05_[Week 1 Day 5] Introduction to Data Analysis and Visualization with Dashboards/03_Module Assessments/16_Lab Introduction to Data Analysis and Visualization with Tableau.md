# Lab: Introduction to Data Analysis and Visualization with Tableau

## ![](https://moringa.instructure.com/courses/1426/files/631398/preview?) Lab: Introduction to Data Analysis and Visualization with Tableau

Now that you have learned of the many features of Tableau, it is time to put that knowledge to work and create some more worksheets and then combine them into a single dashboard. You will create several Tableau worksheets and generate basic visualizations that will then be used to create a simple dashboard.

We will be using a dataset regarding historical Olympics data stretching from 2022 - 1896. The dataset is from Kaggle. You will need to follow the link provided in Tools and Resources and download the data files to complete this lab. The two most relevant files to use will be **olmypic\_medals.csv** and the **olympic\_hosts.csv** which should be connected via a relationship in Tableau using the Slug Game and Game Slug fields respectively. You are welcome to use the other files as well if that fits with the visuals you want to create.

**Scenario:** You are a junior analyst working for the International Olympic Committee (IOC) and have been tasked with creating a dashboard that represents the historic results of the Summer Olympics. This dashboard will be used to promote the next upcoming Summer Olympics. You have freedom of choice in what metrics to look at and create visuals for.

Your dashboard should include at **minimum three separate visuals** and it needs to incorporate some form of **interactivity based on a filtering action**.

Recall the Dashboard creation process:

1. Plan your Dashboard.
2. Create your individual visualizations.
3. Create a new Dashboard.
4. Choose your layout.
5. Add sheets.
6. Arrange and resize sheets.
7. Add filters.
8. Add titles and text.
9. Customize the appearance.
10. Publish your dashboard.

### Instructions

1. Connect to the data downloaded from Kaggle via Data source

* You will need to specify table relationships based on SLUG GAME/GAME SLUG fields.

1. Plan out your dashboard and what questions/visualizations you will include

* Keep in mind that this should focus only on the summer olympics, make sure to create a filter for just summer, use the olympic\_hosts file.

1. Create individual visualizations in separate worksheets

You have free reign on what information you think is relevant to showcase and could take this in many directions. Here are a few provided suggestions to start with but you do not have to use these.

* + Number of Unique events across Years
  + Number of Events per Discipline
  + Number of participants across Years
  + Breakdown of medals by event or discipline
  + Gender representation across Years
  + Gender representation across events or discipline
  + Country representations across Years
  + Country medal count breakdowns

1. Create dashboard and design a rough layout of containers

* Use the default view and generic desktop size
* Think heavily about the F-pattern when planning and the designing the layout

1. Populate dashboard with your individual visualizations

* Make sure to include legends when and where appropriate
* Rearrange, resize, reformat as necessary to form a cohesive dashboard

1. Add in interactivity via user filtering

* Either chart based filtering or menu based filtering selections, or both
* Think about what to filter on

1. Titles, text, and customization

* Incorporate any other elements you see fit into the dashboard
* Make sure to include a logo for the Olympics!

1. Save and Publish

* Once you have a completed dashboard go ahead and save and publish it to Tableau Public.
* Submit the link to your dashboard for this assignment.

Consider posting your dashboard to LinkedIn, social media, and/or adding to your portfolio to start highlighting your work!
