### Terms Recap

* **Dashboard:** A visual display of key information consolidated on a single screen.
* **Layout:** The arrangement of visual elements on a dashboard.
* **Color Theory:** The study of how colors interact and impact user perception.
* **KPI (Key Performance Indicator):** A measurable value that demonstrates how effectively a company is achieving key business objectives.

### Key Concepts

* **F-Pattern Layout:** Organizing information based on natural eye movement.
* **Color Theory:** Using colors to convey meaning and guide attention.
* **Chart Selection:** Choosing appropriate chart types for different data and purposes.

### Brief Process Overview

1. **Define Dashboard Objectives:** Identify key questions and target audience.
2. **Gather and Prepare Data:** Information needed to answer questions.
3. **Define Metrics and KPIs:** How to answer questions.
4. **Select Visuals:** Appropriate chart types for the right data.
5. **Design Layout:** Arrange elements following the F-pattern and utilize color theory to select appropriate color patterns.
6. **Develop Dashboard:** Implement data visualization principles to create dashboard.
7. **Deploy and Maintain:** Launch dashboard so it can viewed.
8. **Document and Refine:** Gather user feedback and iterate on the design.
