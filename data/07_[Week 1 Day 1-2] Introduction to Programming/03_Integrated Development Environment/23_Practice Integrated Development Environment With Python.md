### Instructions

### Step 1: Setup - Running Python Code from Terminal

Before working directly in VS Code, ensure you can run Python scripts from the terminal:

1. **Open a Terminal or Command Prompt:**

* On Windows: Press Win + R, type `cmd`, and press Enter
* On macOS: Open Spotlight Search (cmd+Space), type `Terminal`, and press Enter.
* On Linux: Open your terminal application.

1. **Navigate to Your Script's Directory:**

* Use the `cd` command to navigate to the directory where your Python script is located.

1. **Run a Python Script:**

* Use the command: `python script.py` or  `python3 script.py python3` `script.py`  to execute your Python script. Replace `script.py` with the actual filename.
* The Python interpreter will run the script and display any output or results in the terminal.

### Step 2: Create Simple Python Script

Let's create a simple Python script that prints "Hello, World!" to the console to practice the basics of writing and executing Python code. 

1. **Open VS Code**
2. **Create a New File:**

* Click on File/New File or press Ctrl+N (Windows/Linux) or Cmd+N (macOS).

1. **Write Your Python Code:**

* Type the following code into the new file:

```
#!/usr/bin/env python3  
print("Hello, World!")
```

* The first line is the shebang line, specifying the Python interpreter to use.
* The `print()` function displays the text *Hello, World!* on the console.

1. **Save the File:**

* Save your file by selecting File/Save or pressing Ctrl+S (Windows/Linux) or Cmd+S (macOS).
* Name the file `hello.py`

### Step 3: Run the Script in Terminal

Now, run the script you just created to see it in action. 

1. **Open the Integrated Terminal in VS Code:**

* Click on Terminal/New Terminal or press Ctrl+` (Windows/Linux) or Cmd+` (macOS).

1. **Navigate to the File's Directory:**

* Use the `cd` command to navigate to the directory where you saved `hello.py`.

1. **Run the Script:**

* Type `python hello.py` or `python3 hello.py` and press Enter.
* You should see the output Hello, World! displayed in the terminal.

You’ve now learned how to create, save, and run a basic Python script using VS Code. You’ve also seen how to use the terminal to execute Python commands. These foundational skills will be crucial as you continue to develop more complex programs throughout this course.

Feel free to revisit these steps as you practice and get comfortable with VS Code and Python. As you become more familiar with this powerful IDE, you’ll find it an indispensable part of your development toolkit.
