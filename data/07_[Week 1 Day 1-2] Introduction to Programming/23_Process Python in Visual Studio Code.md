As we move deeper into programming, it is essential to have Python and an Integrated Development Environment (IDE) ready for use. We will use Visual Studio Code (VS Code) as our IDE of choice. This setup will allow you to write and execute Python scripts seamlessly.

First, make sure that you have Python downloaded and installed correctly for your operating system by following these steps:

### [Step 1: Download Python](#dpPanel0)

1. **Go to the Python Downloads Page**

* + Open your web browser and navigate to the [Python Downloads page


    Links to an external site.](https://www.python.org/downloads/ "Link").

1. **Select Your Operating System**

* + The website usually detects your operating system and suggests the most appropriate download at the top of the page.
  + To choose a different version or operating system, scroll down to *Looking for a specific release?*and select the latest stable version.

1. **Download the Latest Stable Version**

* + Click the button: **Download Python 3.x.x**  (where 3.x.x is the latest version number).
  + This will start downloading the installer for your operating system.

### [Step 2: Install Python](#dpPanel1)

1. Go to your Downloads folder and find the Python installer file.
2. Look for the installer that matches your operating system and system architecture.
3. Double click the installer file to launch it and start the installation process.

### [Step 3: Installation Wizard](#dpPanel2)

**On Windows:**

1. During the installation, check the box to add Python to the PATH environment variable. This makes it easier to run Python from the command prompt.
2. Choose your preferred installation options (such as installing for all users or just for the current user).
3. Click "Install" to proceed with the installation.

**On macOS:**

1. Double-click the .pkg file to open the installation wizard.
2. Follow the on-screen prompts, clicking *Continue* as needed.
3. After the installation completes, you can safely move the installer to Trash.

**On Linux:**

1. Python often comes pre-installed on many Linux distributions. To check, open a terminal and type `python` or `python3`.
2. If Python is not installed, use your distribution's package manager to install it (e.g., `apt` on Ubuntu, `yum` on CentOS).
3. Follow the instructions from your package manager to complete the installation.
4. Python should now be available from the terminal.

### [Step 4: Verify the Installation](#dpPanel3)

1. Open a command prompt (Windows) or terminal (macOS/Linux).
2. Type `python`or `python3`and press Enter.
3. You should see the Python interactive shell with a prompt `(>>>)`, indicating Python is installed correctly.
4. To exit the Python shell, type `exit()` and press Enter or press Ctrl+D.
