Review the steps for how to select data with SQL:

1. 1. Establish a connection with a SQL database.
   2. Write a query using SQL.
   3. SQL queries follow the mnemonic Sweaty Feet Will Give Horrible Odors.  
        
      1. Select
      2. From
      3. Where
      4. Groupby
      5. Having
      6. Order By
   4. Execute the query.
   5. Get data back.
