The Matplotlib process can be conceptualized as a series of steps that you follow to create visualizations using the library. Take a minute to review this series of steps:

1. Import Libraries.
2. Choose Plot Type based on your variable type.
3. Define Plot Elements - For example .hist(), .scatter(), .bar(), .plot().
4. Customize Appearance - add labels, titles, colors, tick marks, etc.
5. Display the Plot: plt.show().

Since Seaborn is built on top of Matplotlib, the process will start with importing Matplotlib:

1. Import Matplotlib and Seaborn.
2. Load in Data.
3. Create plot in Seaborn using the appropriate method ([Seaborn Documentation


   Links to an external site.](https://seaborn.pydata.org/api.html)).
4. Enhance/Customize the plot (colors, labels, etc.).
