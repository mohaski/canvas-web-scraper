Review the following steps for using Pandas Series and Dataframes:

1. Import data using pandas data loader commands.
2. Inspect the resultant dataFrame using the .info(), .summary(), .head() methods. Use this overview to aid you in identifying cleaning tasks.
3. Drop columns you do not need.
4. If needed, clean up column names.
5. Transform any columns (i.e., bin data, change numeric column to boolean, etc.).
6. Select and filter data subset as required.
