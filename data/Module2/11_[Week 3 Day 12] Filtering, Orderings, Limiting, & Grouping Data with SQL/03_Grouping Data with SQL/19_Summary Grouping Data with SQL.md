### Key Terms and Concepts

* **GROUP By Clause:** Used to group rows that have the same values in specified columns.
* **Aggregate Functions (AVG, COUNT, MAX, MIN, SUM):** Can be applied to each group to calculate summary statistics.
* **HAVING Clause:** Filter groups based on the results of aggregate functions. It is applied after the groups have been formed and aggregated.

Grouping data in SQL enables us to:

* Aggregate and summarize with calculated summary statistics like mean and median
* Simplify complex data into more manageable subsets
* Deliver more powerful analysis as we can spot group-wide patterns
