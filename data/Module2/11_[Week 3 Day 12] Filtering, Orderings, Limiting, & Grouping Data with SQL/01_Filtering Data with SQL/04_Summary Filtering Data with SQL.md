### Terms Recap

* **SELECT:** Specifies the columns (fields) that you want to retrieve from the database.
* **FROM:** Indicates the table or tables from which you want to retrieve data.
* **WHERE:** Filters the rows returned based on specified conditions.

### Key Concepts

* Working with ALL the data that an organization possesses locally is a tremendous drain on system resources. It is better by far to only get the data you know you need.
* The WHERE statement in SQL allows analysts to filter data right at the source
* Using additional operators (AND) enables an analyst to filter on multiple conditions, not just one.
