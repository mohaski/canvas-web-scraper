### Key Terms and Concepts

* Although we can do a lot of filtering and limiting after a query has been returned, it is still best practice to do as much as can be done in the query itself.
* **ORDER BY:** Enables us to sort on one or multiple columns in the query.
* **LIMIT:** Reduces the number of records returned.
