1. Create a visual using the proper plot type for the variables(s)/data.
2. After creating the visual ask yourself the following:
   1. Do I have x and y labels?
   2. Can I read the labels?
   3. Did I use an accessible and appropriate color scheme?
   4. Am I using annotations to call out main takeaways?
3. If you answered yes to all of the questions in step 2 above you are on your way to implementing great visuals.
