### 3D Visual Example

**Problem:**

This 3D image could use some improvements. It is difficult to judge exactly how tall the individual bars are, and it is also difficult to make direct comparisons.

![Mortality rates in Virginia in 1940, visualized as a 3D bar plot](https://moringa.instructure.com/courses/1406/files/624787/download)

**Solution:**

Use a 2D version with four panels instead.

![Mortality rates in Virginia in 1940, visualized as a Trellis plot.](https://moringa.instructure.com/courses/1406/files/624991/download)

### 2D Visual Example

**Problem:**

This 2D image could use some improvements. This visual is too busy. It contains a lot of visual noise, and the different point types do not strongly separate from each other.

![City fuel economy versus engine displacement, for cars with front-wheel drive (FWD), rear-wheel drive (RWD), and all-wheel drive (4WD).](https://moringa.instructure.com/courses/1406/files/625044/download)

**Solution:**

Drawing the same figure with solidly colored shapes addresses this issue.

![City fuel economy versus engine displacement, for cars with front-wheel drive (FWD), rear-wheel drive (RWD), and all-wheel drive (4WD).](https://moringa.instructure.com/courses/1406/files/625055/download)
