### Terms Recap

* **Numeric Variables:** Data representing quantities that can be measured with numbers (for example, weight, age, and price).
* **Discrete Numeric Variables:** Whole numbers you can count (for example, customer visits per day).
* **Continuous Numeric Variables:** Can take any value within a range (for example, product weight).
* **Categorical Variables:** Data categorized into distinct groups or labels ( for example, product category: clothing, electronics).

### Key Concepts

* The distinction between numeric and categorical variables is crucial for choosing appropriate data visualization techniques.
* **Discrete data** is best represented with bar charts or histograms to show frequencies of each category.
* **Continuous data** is often visualized with histograms or scatter plots to reveal trends and distributions.
* Effective data visualization requires understanding the data types and selecting the most suitable chart for clear communication of insights.

### Brief Process Overview

1. **Identify the data type:** Understand if your data is numeric (measurable with numbers) or categorical (grouped into distinct labels).
2. **Distinguish between discrete and continuous numeric variables:** If numeric, determine if the data represents whole numbers you can count (discrete) or values within a range (continuous).
3. **Choose the appropriate visualization technique:** Based on the data type (numeric vs. categorical, discrete vs. continuous), select the chart that best represents the data (e.g., bar charts for discrete data, histograms or scatter plots for continuous data).
