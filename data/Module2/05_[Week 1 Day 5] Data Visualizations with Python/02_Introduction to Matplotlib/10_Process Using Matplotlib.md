To use Matplotlib, follow these steps:

1. Import Libraries.
2. Choose Plot Type based on your variable type.
3. Define Plot Elements - for example .hist(), .scatter(), .bar(), .plot()
4. Customize Appearance - add labels, titles, colors, tick marks, etc.
5. Display the Plot : plt.show()
