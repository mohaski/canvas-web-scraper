### Terms Recap

* **Matplotlib:** A fundamental Python library for creating various data visualizations (static, animated, interactive).
* **Plots:** Visual representations of data using elements like lines, bars, and markers.
* **Scatter Plot:** Shows relationships between two numeric variables using data points.
* **Histogram:** Depicts the distribution of a single numeric variable.
* **Bar Chart:** Represents categorical data or frequency of numeric data using bars.
* **Line Graph:** Ideal for visualizing trends or relationships over time (numeric x-axis).

### Key Concepts

* Matplotlib offers a versatile toolkit to transform raw data into clear and informative visualizations.
* Choosing the right plot type depends on the data (numeric vs. categorical) and the relationships you want to explore.
* Customization options allow tailoring visualizations to match specific needs and presentations.

### Process Overview

1. **Import Libraries:** Load matplotlib.pyplot for plotting functionalities.
2. **Choose Plot Type:** Select the appropriate plot (scatter, histogram, bar, line) based on your data and analysis goals.
3. **Define Plot Elements:** Use Matplotlib functions like plot(), bar(), hist() to create the visual elements.
4. **Customize Appearance:** Adjust labels, titles, colors, grid lines, etc., to enhance clarity.
5. **Display:** Use plt.show() to display the plot.
