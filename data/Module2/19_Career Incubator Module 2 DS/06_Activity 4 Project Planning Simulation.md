# Activity 4: Project Planning Simulation

## **Introduction**

Data science projects fail at the planning stage far more often than at the technical stage. Underestimating data cleaning time, not identifying stakeholders early enough, and failing to define 'done' are three of the most common project killers. This activity gives you practice structuring a realistic project plan for an analytical project before the technical work begins.
