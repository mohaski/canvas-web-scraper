Installing Git involves downloading the appropriate installer for your operating system (Windows, macOS, or Linux) from the official Git website. Once downloaded, running the installer typically guides you through a series of straightforward steps, allowing you to customize settings such as installation directory and preferred text editor.

Watch the video below for a walkthrough on installing Git.

[VIDEO LINK](https://player.vimeo.com/video/989668653?h=09047f488c)

Here is an overview of the install steps.

1. Identify your operating system (MacOS or Windows).
2. Download the appropriate file from [Git


   Links to an external site.](https://git-scm.com/).
3. If using a Windows machine, open the .exe file that was downloaded.  
   1. Follow the instructions provided by [Git


      Links to an external site.](https://git-scm.com/).
   2. Confirm your installation.
4. If using a Mac, install the package from the App Store.  
   1. Follow the instructions provided by Git.
   2. Confirm your installation.
