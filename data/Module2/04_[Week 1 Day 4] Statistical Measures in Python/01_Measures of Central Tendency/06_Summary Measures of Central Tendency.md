In this lesson, we looked at three measures that can be used to identify the central tendency of a given dataset: the mean, the mode, and the median.

**Mean (average):** This is the most familiar term. It's calculated by adding all the values in a dataset and dividing by the number of values. Think of it as the balancing point where all the data would rest if placed on a seesaw. Outliers can significantly affect the mean.

**Median:** This is the "middle" value when the data is arranged in order from least to greatest. If you have an even number of values, the median is the average of the two middle numbers. The median is less sensitive to outliers compared to the mean.

**Mode:** This is the most frequent value in the data set. You can have one mode, multiple modes (bi-modal, tri-modal), or no mode at all if all values appear the same number of times. The mode is useful for identifying the most common category, but it doesn't necessarily represent the center of the data.

These measures will be used throughout our data analysis journey and, with practice, we will learn to see how we can choose one (or more) of these measures to represent different datasets with different characteristics.
