1. Load CSV file into desired nested data structure (list of list or list of dicts) using Python File I/O and either the csv module’s reader or DictReader.
2. Convert data in a given column to an appropriate data type.
3. Clean and transform data as necessary for each column. The particular tasks will depend on data type, overarching data schema, and the problem at hand.
4. Once cleaning of all columns have been completed, use the csv module’s writer or DictWriter to save your data contained in a respective Python data structure to CSV file.
