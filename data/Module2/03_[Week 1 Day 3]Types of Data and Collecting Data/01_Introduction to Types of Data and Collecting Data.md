### Example: Understanding Data Types in Retail Analysis

![Coats on a display rack in a store window](https://moringa.instructure.com/courses/1406/files/625048/download)

To illustrate the importance of data types in data science, let’s consider an example involving a junior data scientist working for a retail company. This role involves analyzing customer purchase data to identify trends and optimize marketing campaigns.

This would mean dealing with both quantitative variables (e.g., total amount spent, number of items purchased) and categorical variables (e.g., customer zip code, product category). Quantitative data, such as the total amount spent, is represented using numerical data types like integers and floats. Categorical data, such as the product category, is represented using strings. Understanding these distinctions helps in accurately cleaning and analyzing the data.

For instance, recognizing that customer\_zip\_code is a categorical variable (nominal) prevents the data scientist from performing meaningless mathematical operations on it, like averaging zip codes. Instead, they focus on methods to uncover how purchasing behavior varies by region, enabling them to target marketing efforts more effectively. This example underscores the critical role of understanding and correctly representing data types in the data science process.

### Learning Outcomes

After completing this lesson, you should be able to:

* Use correct Python data types to express different classes of data.
* Identify relevant string cleaning tasks in structured or unstructured text data.
* Clean and transform discrete and continuous numeric data using Python’s built-in math functions and logical comparison operators.
* Load and parse CSV files encoding tabular data for further wrangling and analysis.
* Load and navigate hierarchical JSONs to extract relevant information for further wrangling and analysis.
* Distinguish between the four main sampling strategies – simple random, stratified, cluster, and sequential sampling – and know what collection methodology is appropriate for their problem.
