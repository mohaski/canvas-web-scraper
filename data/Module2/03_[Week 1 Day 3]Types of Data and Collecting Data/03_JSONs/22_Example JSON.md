### Marketing Company

**Problem****![X app details on a cell phone](https://moringa.instructure.com/courses/1406/files/625265/download)**

Marketing companies rely on real time ingestion, wrangling, and analysis of posts and trends on social media platforms for tailored and targeted marketing. As a junior data engineer for a particular marketing company, you have been tasked with monitoring data coming in from X (formerly Twitter) and Facebook. This data comes structured in JSON format. X has just rolled out a new application programming interface and a senior data engineer on your team has had to adjust the data extraction from the platform. The senior engineer wants you to investigate how the data is structured in the JSONs being extracted after the change. You are also being asked to amend any functions as necessary in the data processing pipeline to ensure that you are grabbing the same data as you were before and that it is structured in the same way.

**Data Science Solution**

Armed with a firm grasp of how to navigate JSON structures, you load the JSON into a Python data structure using the json module, systematically using your Python data wrangling skills to uncover the structure of the new JSON. You then use this insight to amend the functions in the data processing pipeline that you have found required some tweaking given the new JSON structure. Your senior engineer reviews your work and greenlights it to be pulled into the main branch of the data pipeline code.
