In this topic, you'll delve deeper into a new text-based format: the JSON file. JSONs, short for JavaScript Object Notation, are one of the most common formats for data extracted from the web. In fact, the importance of JSONs cannot be overstated – they form the backbone of modern web interactions. You will encounter these frequently when pulling data from web-based applications (e.g., Yelp, Google maps, etc.) or when you see some data that resides on a website and want to extract that information programmatically (a technique known as scraping). 

JSON has a structure that is a bit different from CSVs. The data is typically nested and hierarchical in a form that does not just have to be tabular in nature. This topic will equip you to understand how JSONs are structured and how they are represented in Python. Through hands-on exercises, you'll learn to load, parse, and manipulate JSON data, enabling you to work effectively with data from web services and applications.
