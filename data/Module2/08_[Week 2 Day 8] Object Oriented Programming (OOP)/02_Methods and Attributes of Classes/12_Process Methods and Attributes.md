Review the following steps to help you use Methods and Attributes correctly.

1. Follow naming conventions for classes (NameLikeThis)
2. What will be the functionality of this class  
   1. Define what methods the class will contain
   2. Define what attributes the class will contain
3. Identify what attributes can be and should be assigned via the \_\_init\_\_ magic method  
   1. When you instantiate an instance object what do you need to provide as arguments
4. Create the class  
   1. Start with defining the \_\_init\_\_ method  
      1. Make sure to assign defaults if needed/wanted
      2. Don’t forgot to include ‘self’ as the first parameter
5. Within your class define methods and assign appropriate attributes  
   1. Remember to always include ‘self’ here for each method
6. Test out your class with different inputs to ensure proper usage
