In this lesson, you'll take a look at class and instance objects in Python and how to create them. A Python class can be thought of as the blueprint for creating a code object (or instance object). It has both the layout for new objects as well as the ability to create those objects. When you initialize or make a new instance object from a class, you are essentially pressing a button on an assembly line that instantly rolls out a new instance object.

For example, if you were dealing with a Car class, you would get a brand new car from the assembly line. In cases where you want to create multiple objects, you can see how this functionality would be extremely useful.
