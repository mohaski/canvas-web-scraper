In this lesson, you learned about what you use classes for and how to define them. Classes are the blueprints for creating instance objects that allow us to create multiple objects with the same or similar attributes and methods. However, all instance objects are produced with unique IDs, making them unique objects.

We instantiate a class by calling class and giving it a name. In this lesson we called:

```
class Ride:  
    pass
```

Class tells Python we want to make a new class object, a new blueprint, called Ride. The pass statement is simply a placeholder until we can assign some attributes and methods to the Ride class.
