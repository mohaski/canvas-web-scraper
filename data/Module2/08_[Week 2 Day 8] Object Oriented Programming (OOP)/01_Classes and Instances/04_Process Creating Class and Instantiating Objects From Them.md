Utilizing OOP and classes within Python is relatively easy and straightforward. In fact, you have already been using classes with many common libraries like matplotlib and especially Pandas, pd.Dataframe is the class that methods like pd.read\_csv() call in order to create new dataframe objects from a particular file. It is also easy to create and use your own custom classes.

1. Creating a class:  
   1. Similar to how functions are defined in Python  

      ```
      class <Class_Name_Here>():  
       # Code for the class
      ```
2. The custom class can then be given attributes and methods; more on this in topic 2 of this module.
3. Instantiating objects from a class  
   1. The class is called and any defining arguments are given during this initialization → more on this in topic 3 of this module.
   2. Unique instances (objects) are created each time the class is called
   3. These objects are typically assigned as variables within Python  

      ```
      instance_object1 = Class_Example(<defining arguments>)  
      instance_object2 = Class_Example(<defining arguments>)
      ```
