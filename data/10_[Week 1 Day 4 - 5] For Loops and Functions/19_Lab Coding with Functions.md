# Lab: Coding with Functions

## ![](https://moringa.instructure.com/courses/1391/files/620338/preview?) Lab: Coding with Functions

A **function** in Python is a reusable block of code that performs a specific task and can be executed when called. Functions are defined using the def keyword, followed by a name, parentheses (which may include parameters), and a colon. The body of the function contains the code to be executed and may include a return statement to send a result back to the caller.

**Functions are used to organize code into logical, manageable pieces, making programs more readable, efficient, and easier to maintain.** Allowing you to avoid repetition by encapsulating tasks that are performed multiple times throughout a program. Functions also support abstraction, letting you think about what a block of code does without worrying about how it does it. This is particularly useful in large projects, debugging, and testing.

In this lab, you are going to be assessed on your ability to:

* Define functions with specified inputs and outputs.
* Call functions.
* Debug function formatting.
* Differentiate between printing and returning output.
