# Lab: Coding with For Loops

## ![](https://moringa.instructure.com/courses/1391/files/620338/preview?) Lab: Coding with For Loops

A **for loop** is a control structure in Python used to iterate over a sequence, such as a list, tuple, dictionary, string, or range of numbers. It executes a block of code once for each item in the sequence. Each iteration temporarily assigns the next value in the sequence to a loop variable, allowing the loop body to operate on it. This structure ensures that code is repeated a specific number of times or until all elements in the sequence have been processed.

**For loops are useful when you know exactly how many times you need to repeat a task, or when you're working with a fixed set of items.** Commonly used for tasks like processing each item in a list, iterating over characters in a string, or performing a specific operation for a defined range of numbers. For example, if you have a list of student names and want to print each one, a **for** **loop** is ideal because it will execute once for each student in the list.
